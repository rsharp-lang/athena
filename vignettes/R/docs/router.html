router

Description

Handles URL routing for web requests, mapping URLs to local R scripts or static files in the web context directory.

Usage

router(url, webContext);

Arguments

url

Parsed URL object from incoming request (should contain $path)

webContext

Base directory path for web resources

Details

Routing logic:\cr - Root path (/) maps to index.html\cr - URLs ending with .R get direct script mapping\cr - Other URLs resolve to static files\cr Ensures proper extension handling for R script execution.

Authors

xieguigang

Value

Returns full filesystem path to either:\cr - Corresponding R script (appends .R if missing)\cr - Static file (defaults to index.html for root path)\cr
