exec_r {Athena} R Documentation

exec_r

Description

Executes R scripts identified by the router and returns generated content through HTTP response.

Usage

exec_r(webContext, req, response);

Arguments

webContext

Base directory containing web resources

req

HTTP request object containing URL information

response

HTTP response object to write output to

Details

Operation flow:\cr 1. Resolve URL to local file path via router()\cr 2. If file exists, execute with source() and stream output\cr 3. Handle missing files with 404 error including normalized path\cr Ensures proper error handling and path disclosure prevention.

Authors

xieguigang

Value

Writes execution results to response object or returns HTTP error
