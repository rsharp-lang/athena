run_http {Athena} R Documentation

@title Configure and Start HTTP Server

Description

Initializes HTTP server with specified port and web context directory, registers request handlers for different HTTP methods.

Usage

run_http(
    httpPort = '80',
    webContext = './wwwroot');

Arguments

httpPort

(character/numeric) Port to listen on. Default: "80".

webContext

(character) Path to web root directory. Default: "./wwwroot".

Details

1. Sets global configurations:\cr - \code{wwwroot}: File system handler for static content\cr - \code{deepseek}: Initialized Ollama AI context\cr - \code{apps}: URL routing rules from \code{scan_urls()}\cr 2. Attaches custom HTTP headers (X-Powered-By, Author, etc.)\cr 3. Registers handlers for GET/POST/PUT methods\cr 4. PUT method returns a test success message by default

Authors

xieguigang

Value

Returns the HTTP socket object. Use \code{listen()} to start server.
