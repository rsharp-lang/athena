proxy_realpath {Athena} R Documentation

Resolve file storage path

Description

Constructs physical file path using hash key segmentation strategy. Files are stored in nested directories derived from hash segments to avoid filesystem overload in high-concurrency environments.

Usage

proxy_realpath(key);

Arguments

key

A 32-character hash string generated by proxy storage system. Must contain at least 25 characters for valid path resolution.

Details

Authors

xieguigang

Value

Character string representing full file path following structure: `/<3-5charsubstr>/<23-25charsubstr>/` Returns empty string if no matching file found.
