init_ollama {Athena} R Documentation

@title Initialize Ollama AI Client

Description

Initializes and configures the Ollama AI client using global options. This function sets up the model and server connection, builds required tools, and returns the initialized client object for subsequent AI operations.

Usage

init_ollama();

Arguments

Details

Uses `ollamamodel` and `ollamaserver` options from global environment to configure the client. Automatically attaches essential tools via `buildollamatools()` during initialization.

Authors

xieguigang

Value

Returns an initialized Ollama client object that's ready for AI operations. The returned object contains configured tools and settings.
